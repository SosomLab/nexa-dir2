# assets — 앱 아이콘 보관

원본 [Nexa Dir](https://github.com/SosomLab/nexa-dir)(`../nexa-dir/app/Nexa.App/Assets/AppIcon/`)에서
가져온 앱 아이콘 원본. 디자인 = 폴더 + 터미널 프롬프트(`>_`) 초록 글리프.

| 파일 | 용도 |
| --- | --- |
| `nexa-dir.ico` | **다크** 앱 아이콘(exe 임베드용 — 다중 해상도 ico) |
| `nexa-dir-light.ico` | **라이트** 앱 아이콘(ico) |
| `nexa-dir-1024.png` | 다크 1024×1024 원본(스토어·문서·재생성 소스) |
| `nexa-dir-light-1024.png` | 라이트 1024×1024 원본 |

- 출처 SSOT = 원본 리포(같은 조직 SosomLab). 디자인 원본 파일(.psd/.svg 등)은
  원본 리포에도 없음 — 위 raster(PNG/ICO)가 최상위 소스.
- 배포 시 exe 아이콘 임베드는 [docs/21-distribution.md](../../../docs/21-distribution.md)
  참조(M5 패키징 — 아이콘/폰트 리소스 임베드).
