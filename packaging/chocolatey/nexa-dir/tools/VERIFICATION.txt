VERIFICATION
------------
Verification is intended to assist the Chocolatey moderators and community in
verifying that this package's contents are trustworthy.

This package embeds no binaries. At install time it downloads the official
installer published by the software vendor (SosomLab, who also maintains this
package) on the project's GitHub Releases page:

  https://github.com/SosomLab/nexa-dir2/releases

The download URL and its SHA-256 checksum are written into
tools\chocolateyinstall.ps1 by the project's release pipeline
(.github/workflows/release.yml), which builds the installer and computes the
checksum in the same GitHub Actions run that publishes the release asset.

To verify the download yourself:

  1. Read $url64 and $checksum64 from tools\chocolateyinstall.ps1.
  2. Download that URL from the GitHub Releases page above.
  3. Compare the checksums:

       Get-FileHash -Algorithm SHA256 .\NexaDir-Setup-<version>.exe

     or, with the Chocolatey checksum utility:

       checksum -t sha256 -f .\NexaDir-Setup-<version>.exe

The file's SHA-256 digest is also shown by the GitHub Releases API for each
asset, independently of this package.
