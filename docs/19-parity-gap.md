- **클립보드 쓰기측**(원본 CLIP 잔여): 앱→탐색기 Ctrl+C/V 양방향
- **nexa-ops 러스트 이관**(원본 B-1 잔여): 전송·삭제·리네임·undo 전부 rlib
- **빈 영역 배경 셸 메뉴 S2**(원본 B-2 잔여) · **포터블 단일 exe·릴리스 파이프라인**(원본 PKG-4 대기)
