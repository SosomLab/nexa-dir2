# 02 · 로드맵 (원안, M0~M5)

> 원본 M1 기능군을 예산(DR-2) 안에서 재현하는 순서. 현황은 [MILESTONES](MILESTONES.md), 순차 백로그는 [TODO](TODO.md).
> 원칙: **뷰어(읽기) → 셸 골격 → 조작(쓰기) → 패널 확장 → 마감** — 각 단계 끝에 예산 게이트.

## M0 — 기반·게이트 (목표: ADR-0001 실증)

- 워크스페이스 스캐폴딩 · 원본 코어 3크레이트(nexa-core/vfs/tree) rlib 이식(테스트 green).
- Win32 창 스켈레톤(`#[cfg(windows)]`) + GDI 렌더 스파이크 · CI(core/windows).
- **게이트**: 빈 창 유휴 RSS·exe 크기·임포트 테이블 실측 → 예산 성립 확인. 태그 `0.1.0`.

## M1 — 뷰어 코어 (★ 플래그십 재현)

- `nexa-gui` 분리(위젯·무효화·입력 라우팅) · ADR-0002 렌더링 확정(DirectWrite interop).
- 가상화 파일 리스트: 컬럼(이름/확장자/수정일시/종류/크기)·헤더 정렬(3상태·다중열)·셸 아이콘(LRU).
- **인라인 폴더 확장 + 교차폴더 다중 선택**(코어 배선)·키보드 네비 전체·타입어헤드·러버밴드.
- 숨김/점 파일 토글 · 네비게이션(뒤로/앞으로/위로·히스토리).
- **게이트**: 100k 첫 렌더 <150ms·스크롤 60fps·RSS 예산 내.

## M2 — 셸 골격·상주

- 계층 경로 바(세그먼트·편집·자동완성) · 좌/우 듀얼 패널 + 패널별 탭(드래그·잠금/고정) · 스플리터.
- 메뉴 바·도구 모음·상태바 · 테마(라이트/다크/시스템, 원본 토큰) · 설정/세션 영속(`data\`) · i18n(내장 en/ko).
- IME(한글) 1차 · UIA 접근성 1차 · 유휴 트림(상주 규율).
- **게이트**: 상주 시나리오(탭 4·듀얼) RSS ≤30MB.

## M3 — 파일 조작 (쓰기)

- `nexa-ops`: 전송 엔진(복사/이동·진행률·충돌 정책·취소)·삭제(휴지통/완전)·이름변경·새로 만들기·**Undo/Redo**.
- `nexa-shell`: 셸 컨텍스트 메뉴(IContextMenu+고유 병합, 원본 ADR-0005)·클립보드 상호운용·OLE DnD.
- watcher(무간섭 갱신 — 원본 NAV-UPFOCUS 규약).

## M4 — 하단 패널

- 정보 뷰 · 내장 미리보기(텍스트/이미지) · `nexa-term`: ConPTY + VT 스크린(원본 VtScreen 이식).

## M5 — 마감·릴리스

- 퀵 런처 · 일괄 이름변경(원본 docs/25) 등 원본 잔여 패리티 · 접근성/IME 마감.
- 릴리스 파이프라인(단일 exe 자동 첨부) · 예산 최종 게이트 · 서명 결정(원본과 공동).
- 후속(원본 동기): 검색(M3원본)·아카이브·클라우드·플러그인 ADR 재검토.
