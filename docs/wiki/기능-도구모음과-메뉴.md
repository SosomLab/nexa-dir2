# 도구 모음과 메뉴

도구 모음, 셸 컨텍스트 메뉴, 퀵 런처 바 — 그리고 이들을 **커스터마이즈**하는 방법을 정리합니다.

---

## 도구 모음 (Toolbar)

그룹화된 SVG 아이콘 버튼(26px 바)입니다. 기본 순서: `새로 고침 | panel 그룹 | view 그룹 | show 그룹 | 설정`. 그룹 사이에는 구분선이 있습니다.

| 그룹 | 버튼 |
| --- | --- |
| (단독) | **새로 고침** (⟳) |
| **panel** | 파일 패널 토글(▌▐) · 하단 도크(▂) · 정보 패널 토글(ⓘ) · 컬럼 너비 동기화(⇔) |
| **view** | 트리(├─) · 플랫(☰) · 타일(▦) — 라디오 |
| **show** | 숨김 파일(👁) · 점 파일(…) |
| (단독) | **설정** (⚙) |

- **SVG 아이콘**: 자체 SVG 서브셋 파서 → GDI+ 래스터. 3상태 색(기본 진회색 · 선택 파랑 `#3D8BFF` · 비활성 알파 38%).
- **툴팁 i18n**: 버튼에 마우스를 올리면 현재 언어의 툴팁이 뜹니다(예: "듀얼 파일 패널" / "Dual File Panels"). 토글 버튼은 상태에 따라 라벨이 바뀝니다.

---

## 도구 모음 / 컬럼 / 컨텍스트 메뉴 커스터마이즈

세 가지 모두 **공통 편집 창**(순서 편집 트리)으로 표시/숨김과 순서를 바꿉니다.

![도구 모음 순서 편집 창](https://raw.githubusercontent.com/wiki/SosomLab/nexa-dir2/images/22-order-editor-toolbar.png)

- **진입**: 설정의 `도구 모음 순서` / `파일 컬럼` / `컨텍스트 메뉴 항목`의 [편집…] 버튼, 또는 도구 모음 빈 영역·컬럼 헤더 우클릭 팝업.
- **편집 규칙**:
  - **체크박스**(맨 앞) = 표시 여부. 그룹 체크를 끄면 하위가 통째로 숨고, 하위 체크는 값을 유지한 채 비활성(회색)됩니다.
  - **셰브론**(▾/▸) = 그룹 펼침/닫음(하위가 있는 그룹만).
  - **순서 이동**: 그룹은 통째로, 항목은 그룹 안에서만. 위/아래 화살표 버튼 또는 **드래그**(고스트 박스·삽입 미리보기·가장자리 자동 스크롤·`Esc` 취소).
  - `Shift`+클릭 = 같은 그룹 내 연속 다중 선택. 키보드 `↑`/`↓`(단일)·`Shift+↑/↓`(다중)·`Ctrl+↑/↓`(순서 이동)·`Space`(체크).

## 셸 컨텍스트 메뉴

파일/폴더를 **우클릭**하면 Windows 셸 컨텍스트 메뉴가 나타납니다(7-Zip·Git·보내기·속성 등 셸 확장 포함).

- **조작**: 우클릭 · `Apps` 키 · `Shift+F10`. 빈 영역 우클릭 = 배경 메뉴(새로 만들기 서브메뉴 등).
- **고유 병합 항목**: 앱 고유 항목이 셸 메뉴에 병합됩니다 — `완전 삭제`, `폴더에 붙여넣기`/`붙여넣기`, `실행 취소`/`다시 실행`, `경로 복사`, `이름 복사`. delete/rename/paste 동사는 앱이 가로채 되돌리기·인라인·전송에 합류시킵니다.
- **순서/표시 설정**: 앱 고유 항목(항목 메뉴/배경 메뉴)의 표시/순서를 위 편집 창에서 바꿉니다(셸 제공 항목은 대상 아님).

## 퀵 런처 바 (Quick Launcher Bar)

도구 모음 아래에 상시 표시되는 런처 바입니다. 자주 쓰는 외부 프로그램을 버튼으로 등록합니다(exe 셸 아이콘 16×16·그룹 구분선).

- **동작**: 클릭하면 `%path%`를 활성 패널의 현재 폴더로 치환한 뒤 실행합니다(작업 디렉터리 동반). 실패는 상태바에 격리 표시.
- **조작/설정**: 보기 메뉴 `퀵 런처 바`로 표시 토글. 항목은 `data\settings.cfg`의 `launcherN=라벨|exe|인자`(구분선=`launcherN=-`)로 정의합니다. VS Code/pwsh/cmd가 기본 시드됩니다. (항목 추가/삭제 UI는 후속 예정.)
