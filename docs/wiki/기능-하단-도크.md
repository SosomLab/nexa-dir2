# 하단 도크

**하단 도크**는 파일 목록 아래에 붙는 전폭 밴드로, **정보 / 미리보기 / 터미널** 세 가지를 담습니다.

![정보 패널이 채워진 하단 도크](https://raw.githubusercontent.com/wiki/SosomLab/nexa-dir2/images/10-dock-info.png)

- **표시 토글**: `` Ctrl+` `` · 보기 메뉴 `하단 도크` · 도구 모음 `dock`(▂) 버튼. 기본 표시.
- **종류 전환**: 도크 좌상단의 종류 스트립(`정보` | `미리보기` | `터미널`)을 클릭.

---

## 정보 (Info)

선택 항목의 메타데이터를 표시합니다.

- 다중 선택 = 개수. 단일 선택 = 이름 · 종류 · 크기(원시 바이트) · 수정 날짜 · 경로.
- 선택이 바뀌면 자동 갱신됩니다.
- 도크 텍스트는 드래그 선택 + `Ctrl+C` 복사가 됩니다.

### 듀얼 / 싱글 정보 패널

- **듀얼 정보**: 좌↔좌·우↔우 독립 표시.
- **싱글 정보**: 전폭을 공유하며 활성 패널을 추종.
- **조작**: 보기 메뉴 `듀얼 정보 패널`/`싱글 정보 패널`, 도구 모음 `info toggle`(ⓘ) 버튼(듀얼 파일 패널 + 도크 켜짐일 때만 활성).
- 싱글 파일 패널 모드에서는 정보 패널이 싱글로 고정됩니다(선호값은 보존, 듀얼 복귀 시 원복).

---

## 미리보기 (Preview)

선택한 파일의 내용을 미리 봅니다.

- **텍스트**: 파일 첫 16KB를 UTF-8로 디코드, 최대 200줄(탭 4칸). 빈 파일은 `(빈 파일)`, 첫 1KB에 NUL 바이트가 있으면 `이진 파일 — 텍스트 미리보기 없음`.
- **이미지**: 9종 확장자(png/jpg/jpeg/bmp/gif/ico/tif/tiff/webp)를 WIC로 디코드해 비율 유지·가운데 렌더(확대 없음·캐시 8건).

> 미리보기는 향후 **Starlark 플러그인**으로 확장될 예정입니다(EXIF 등). 현재는 내장 텍스트/이미지 미리보기만 동작합니다 — → [플러그인 개발](개발-플러그인).

## 터미널 (Terminal)

내장 ConPTY 터미널입니다. 별도 프로그램 없이 현재 위치에서 셸을 띄웁니다.

- **셸**: pwsh → powershell → cmd 순으로 폴백.
- **기능**: VT 파서(SGR 색·CSI)·스크롤백 800줄·전각(2폭) 셀·마우스 드래그 선택(가장자리 자동 스크롤)·스크롤백 휠·`Ctrl+C/V`·셸 종료 시 아무 키로 재시작·리사이즈 동기.
- **관련 설정**(설정 → 터미널): `터미널 글꼴`(쉼표 = 폴백 체인)·`크기(8~32)`, `줄 바꿈`(끄면 고정 열 + 가로 스크롤), `열 수(80~1000, Shift+휠 가로 스크롤)`.

> 터미널 탭을 열면 셸 프로세스(cmd/pwsh)가 기동됩니다. 이는 정상 동작이며, 도크를 끄거나 다른 종류로 전환하면 유휴 상태가 됩니다.

---

## 레이아웃 · 스플리터

- 도크 높이는 경계(±3px)를 드래그해 조절(비율 0.15~0.5·양 패널 동기).
- 도크 안의 좌/우 분할도 독립 스플리터로 조절합니다(파일 패널 좌/우 비율과 별개).
- 영속: `dock`(표시)·`dock_ratio`(높이)·`dock_split`(좌우 분할).

## 로드맵 — 좌측 도크

향후 **좌측 도크**(디렉토리 트리 내비게이터)가 추가될 예정입니다. 전체 디렉토리 트리를 좌측 전고 밴드에 표시하고, 행을 클릭하면 활성 탭의 위치를 이동시키는 직관적·구조적 탐색 도구입니다(설계 등록 상태).
