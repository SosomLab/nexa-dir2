# 키보드 단축키

Nexa Dir는 키보드 우선으로 설계됐습니다. 전체 단축키 총람입니다.

## 전역 (파일 목록 포커스)

| 단축키 | 동작 |
| --- | --- |
| `F5` | 새로 고침(현재 폴더 재로드) |
| `F6` | 테마 순환(다크 → 라이트 → 시스템) |
| `Tab` | 활성 패널 전환(좌 ↔ 우) |
| `Ctrl+Tab` | 활성 패널의 다음 탭 |
| `Ctrl+T` | 새 탭 |
| `Ctrl+W` | 탭 닫기 |
| `Ctrl+A` | 전체 선택 |
| `Ctrl+C` | 복사(도크 텍스트 선택이 있으면 그 텍스트 우선, 없으면 파일) |
| `Ctrl+X` | 잘라내기(대기 항목은 목록에서 흐리게 표시) |
| `Ctrl+V` | 붙여넣기(폴더 1개 선택 = 그 폴더 안·파일 1개 선택 = 그 파일이 있는 폴더·그 외 = 현재 폴더. 잘라내기는 1회성) |
| `Ctrl+Z` | 실행 취소 |
| `Ctrl+Y` / `Ctrl+Shift+Z` | 다시 실행 |
| `Enter` | 캐럿 행 열기/실행(폴더 진입·파일은 연결 프로그램) |
| `F2` | 인라인 이름 변경 |
| `Del` | 휴지통 삭제 |
| `Shift+Del` | 완전 삭제(확인창) |
| `Ctrl+Shift+N` | 새 폴더 |
| `Ctrl+Shift+R` | 일괄 이름 변경 창 |
| `Ctrl+H` | 숨김 파일 표시 토글 |
| `Ctrl+.` | 점(.) 파일 표시 토글 |
| `Ctrl+,` | 설정 창 |
| `` Ctrl+` `` | 하단 도크 토글 |
| `Apps` 키 | 캐럿 행 셸 컨텍스트 메뉴 |
| `Esc` | 상황별 취소(컬럼 드래그 → 전송 취소 → 열린 메뉴 닫기 → 경로 편집 취소) |

## Alt 조합

| 단축키 | 동작 |
| --- | --- |
| `Alt+←` | 뒤로(히스토리) |
| `Alt+→` | 앞으로(히스토리) |
| `Alt+↑` | 상위 폴더로 |
| `Alt+↓` | 캐럿 행 활성화(더블클릭 동등) |
| `Shift+F10` | 캐럿 행 셸 컨텍스트 메뉴 |

## 캐럿 내비게이션 (파일 목록)

| 키 | 동작 |
| --- | --- |
| `↑` / `↓` | 캐럿 이동 + 단일 선택 + 스크롤 추적 |
| `PageUp` / `PageDown` / `Home` / `End` | 페이지·처음·끝 이동 |
| `Shift`+이동 | 앵커 기준 범위 선택 |
| `Ctrl`+이동 | 선택 변경 없이 캐럿만 이동 |
| `Space` | 캐럿 행 선택 토글(접두사 입력 중이 아닐 때) |
| `Ctrl+Space` | 캐럿 행 선택 토글(캐럿만) |
| `→` | 폴더 인라인 펼침 / 자식으로 |
| `←` | 접힘 / 부모로 |

## 마우스

| 조작 | 동작 |
| --- | --- |
| 마우스 X1 / X2 버튼 | 뒤로 / 앞으로 |
| 행 더블클릭 | 열기/실행 |
| 삼각형(▸/▾) 클릭 | 펼침/접힘(선택과 분리) |
| 본문 클릭 | 선택(`Ctrl` 토글·`Shift` 범위) |
| 빈 영역 드래그 | 러버밴드 다중 선택 |
| 탭 더블클릭 | 설정 동작(기본 닫기 / 고정 / 잠금) |
| 탭 바 빈 공간 더블클릭 | 새 탭 |
| 컬럼 헤더 클릭 / `Shift`+클릭 | 정렬 3상태 순환 / 다중열 정렬 |
| 컬럼 헤더 드래그 | 컬럼 재배열(고스트·미리보기·`Esc` 취소) |
| 컬럼 경계 더블클릭 | 내용 맞춤 auto-fit |
| `Shift`+휠 | 가로 스크롤 |
