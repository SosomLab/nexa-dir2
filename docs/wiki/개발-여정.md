# 개발 여정

**단 9일**(2026-07-11 ~ 07-19)의 집중 개발로 M0~M5 마감 + UX 고도화까지 도달했습니다. 각 마일스톤은 "완료 = 테스트 green + 예산 게이트 통과"로만 인정됩니다.

> 예산 게이트가 무엇인지, DR/ADR이 무엇인지는 **[설계 결정](설계-결정)** 문서를 참고하세요.

---

## 타임라인

### M0 — 기반·게이트 → `0.1.0` (07-11)

프로젝트 발족. 원본 스택 한계 실측 → 올 러스트 재구축 결정 후 설계 문서 세트를 확정했습니다. 워크스페이스 스캐폴딩(정적 CRT·LTO·panic abort) → **nexa-core/vfs/tree 원본 이식**(rlib 직접 링크로 FFI/ABI 소멸) → Win32 창 스켈레톤 → GDI 렌더 스파이크(더블 버퍼·100k 행 중 가시 행만).

- **게이트 실측**: 빈 창 유휴 RSS **13.22MB** · exe **0.20MB** · 임포트 6종 전부 인박스.
- 원본 60MB+ → 13MB로 스택 전환의 정당성을 첫날 증명.
- 특기: **맥에서 windows 타깃 `cargo check` green** — WinUI 시절 불가능하던 UI 코드 사전 타입 검증.

### M1 — 뷰어 코어 → `0.2.0` (07-12)

원본 뷰어 축을 순차 재현. `nexa-gui` 분리 → **ADR-0002 렌더링 확정**(DirectWrite interop, GDI 대비 −28%) → 가상화 파일 리스트 → 5컬럼 시스템(정렬 3상태·Shift 다중열) → **★ 플래그십**(삼각형=펼침 vs 본문=선택 분리·**폴더 교차 다중 선택**·러버밴드) → 키보드 네비 + 타입어헤드 → 셸 아이콘(LRU 캐시) → 네비게이션(히스토리·숨김 토글).

- **게이트**: 100k 첫 렌더 **115ms** · 스크롤 2,098µs/프레임(60fps 예산의 13%) · 10k 유휴 27.87MB.

### M2 — 셸 골격 → `0.3.0` (07-13)

탐색기 크롬 완성. 경로 바(브레드크럼 + 편집) → **듀얼 패널 + 패널별 탭** → 메뉴/도구/상태바(**커스텀 드로잉·자식 HWND 0**) → 테마 시스템(시스템/라이트/다크·DWM 다크 타이틀바) → 설정/세션 영속 → i18n(en/ko·동적 전환) → **상주 규율**(유휴 트림: 활성 26.9MB → 유휴 0.21MB) → IME/UIA 1차.

- **게이트**: 듀얼·탭4 상주 **26.9MB** ≤30 통과.

### M3 — 파일 조작 → `0.4.0` (07-13)

"뷰어 → 탐색기" 정체성 완성. 전송 엔진(순번 복제·충돌·4MB 청크·취소) → 삭제/이름변경/새로 만들기 → **Undo/Redo**(휴지통 복원 포함) → **셸 컨텍스트 메뉴**(comctl32 회피) → 클립보드 상호운용 + **OLE 드래그앤드롭** → 폴더 watcher(무간섭 재로드).

- **게이트**: 활성 32.57 → 300초 후 **6.29MB** ≤30.

### M4 — 하단 도크 → `0.5.0` (07-14)

하단 패널 3종. 정보 뷰 → **내장 미리보기**(텍스트·WIC 이미지) → **ConPTY 터미널**(VT 파서·SGR·스크롤백·pwsh→cmd 폴백). 이어 실기 QA로 이동 프리즈 2건을 근본 해소(12,009ms → 4ms).

- **게이트**: 터미널 상주 최악 케이스에서 300초 후 **5.07MB** ≤30.

### M5 — 마감·릴리스 → `0.6.0` (07-15)

퀵 런처 바 → 일괄 이름 변경 α(원본 스펙 **최초 구현**) → **릴리스 파이프라인**(태그 push → 테스트 → 예산 게이트 → GitHub Release 자동 첨부) → 접근성·IME 마감 + **무서명 유지 확정**.

- **게이트**: 유휴 RSS 중앙값 **16.86MB** · exe **0.90MB**.

### 07-16 ~ 07-19 — UX 고도화 → `0.7.0`

마감 이후 사용자 주도 UX 심화. **배포 2채널**(포터블 + Inno Setup 설치형) → 가상 최상위 "내 PC" → 보기 모드 3종(타일/일반/트리) → 폰트 슬롯 → 패널/정보 싱글·듀얼 모드 → **일괄 이름 변경 v2**(카드 스택 파이프라인·정규식·날짜 토큰) → **Nx 커스텀 컨트롤 라이브러리**(14종·GDI+ AA 백엔드) → **SVG 아이콘 파이프라인**(서브셋 파서·9종 전부 SVG) → 컬럼 드래그 재배열·auto-fit → 하단 도크 버튼·용어 체계 확정.

- **게이트**: exe **1.40MB** · 임포트 인박스 17종.
