## 화면 구성

위 화면을 위에서부터 아래로 보면:

1. **타이틀 바** — 현재 활성 패널·경로·항목 수·성능 지표(평균 페인트·첫 렌더).
2. **메뉴 바** — 파일 / 편집 / 보기 (커스텀 드로잉).
3. **도구 모음** — SVG 아이콘 버튼(새로 고침·패널/도크 토글·보기 모드·숨김 토글·설정). → [도구 모음과 메뉴](기능-도구모음과-메뉴)
4. **퀵 런처 바** — 사용자 정의 외부 프로그램 버튼. → [도구 모음과 메뉴](기능-도구모음과-메뉴)
5. **탭 바** (패널별) — 독립 탭 + [+] 버튼. → [패널과 탭](기능-패널과-탭)
6. **경로 바** (패널별) — 브레드크럼 + 편집 모드. → [탐색](기능-탐색)
7. **파일 목록** (좌/우 듀얼) — 5컬럼·트리/플랫/타일 보기. → [파일 목록과 보기](기능-파일-목록과-보기)
8. **하단 도크** — 정보 / 미리보기 / 터미널. → [하단 도크](기능-하단-도크)
9. **상태 바** — 항목 수·선택 수·필터 상태.

## 기능 지도

### [파일 목록과 보기](기능-파일-목록과-보기)
보기 3종(트리/플랫/타일)·가상화 리스트·5컬럼 시스템·다중열 정렬·컬럼 드래그 재배열·auto-fit·인라인 펼침·컬럼 너비 동기화.

### [탐색](기능-탐색)
경로 바(브레드크럼+편집·자동완성)·뒤로/앞으로/위로·히스토리·가상 "내 PC" 루트·드라이브·타입어헤드 검색.

### [패널과 탭](기능-패널과-탭)
싱글/듀얼 파일 패널·패널별 탭·잠금/고정/복제·세션 복원.

### [파일 조작](기능-파일-조작)
복사/이동/삭제/이름변경/새로 만들기·진행률·충돌 처리·Undo/Redo·휴지통·클립보드·드래그앤드롭·폴더 watcher.

### [하단 도크](기능-하단-도크)
정보 패널(듀얼/싱글)·미리보기(텍스트/이미지)·ConPTY 터미널·레이아웃/스플리터.

### [도구 모음과 메뉴](기능-도구모음과-메뉴)
도구 모음(SVG 아이콘·툴팁 i18n·순서/표시 편집)·셸 컨텍스트 메뉴·퀵 런처 바.

### [일괄 이름 변경](기능-일괄-이름변경)
동작 파이프라인 6종(치환/정규식/대소문자/삽입/연번/날짜)·스코프·실시간 미리보기·프리셋.

### [설정](기능-설정)
카테고리별 전체 옵션·검색·즉시 적용·테마·언어·글꼴 슬롯.

### [키보드 단축키](키보드-단축키)
전역·Alt·캐럿 내비게이션·마우스 조작 총람.
