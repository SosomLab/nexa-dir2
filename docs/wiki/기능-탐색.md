# 탐색

폴더 사이를 오가는 모든 방법을 정리합니다.

---

## 경로 바 (브레드크럼 + 편집)

각 패널 상단에는 경로 바가 있습니다.

- **브레드크럼 모드**: 경로가 세그먼트로 표시됩니다. 세그먼트를 클릭하면 해당 폴더로 이동(현재 세그먼트는 비활성·hover 강조).
- **편집 모드**: 경로 바를 우클릭하거나 진입하면 텍스트 편집으로 전환됩니다.
  - `Enter` 제출 · `Esc` 취소(자동완성 팝업이 열려 있으면 팝업만 닫기)
  - `↑`/`↓` 자동완성 항목 이동
  - `Ctrl+C/X/V` 표준 편집 클립보드
  - IME 한글 조합 창 캐럿 위치 지원
- **자동완성**: 입력에 따라 경로 제안 드롭다운이 표시됩니다.
- **shell: 스킴**: `shell:이름` 형태의 특수 폴더 경로를 지원합니다(SHParseDisplayName 위임 — KnownFolders·GUID 자동 커버).

## 뒤로 / 앞으로 / 위로 · 히스토리

| 조작 | 동작 |
| --- | --- |
| `Alt+←` / 마우스 X1 버튼 | 뒤로 (히스토리) |
| `Alt+→` / 마우스 X2 버튼 | 앞으로 (히스토리) |
| `Alt+↑` | 상위 폴더로 |
| 패널별 내비 바 ←→↑ 버튼 | 뒤로/앞으로/위로 (활성/비활성 흐림 표시) |

- **히스토리**: 탭별 독립 히스토리. 새 이동을 push하면 앞으로 스택이 절단됩니다.
- **관련 설정**: `위로 이동 시 선택 위치`(상단/중단/하단) — `Alt+↑`로 상위 폴더로 이동했을 때, 방금 떠난 폴더를 뷰의 어디에 둘지 결정합니다.

---

## 가상 "내 PC" 루트 · 드라이브

드라이브 루트(예: `C:\`)에서 `Alt+↑`를 누르면 **가상 "내 PC" 뷰**로 진입해 드라이브 목록을 표시합니다(드라이브 간 이동).

- 타일 보기에서는 드라이브 용량 바, 상세 보기에서는 `전체 크기`·`사용 가능한 공간` 컬럼이 표시됩니다.
- 드라이브 행을 `Enter`/더블클릭하면 그 드라이브 루트로 이동합니다.
- 드라이브 루트 탭은 `D:` 형태로 간결하게 표기됩니다(후행 `\` 제거).

## 타입어헤드 검색

목록에서 접두사를 타이핑해 항목으로 점프합니다.

- **조작**: 목록에 포커스한 채 문자를 입력. 반복 입력 시 같은 접두사의 항목을 순환. 화면에 HUD 배지가 뜹니다. `Backspace`로 마지막 글자 삭제(옵션). 입력이 멎으면 접두사 리셋(기본 1초).
- **관련 설정** (설정 → 파일 목록 → 타입어헤드):
  - `검색 범위` — 전체 처음부터 / 현재 계층(형제) / 가시 스트림(기본)
  - `입력 리셋(ms)` — 200~10000
  - `배지 위치` — 3×3 그리드 중 선택
  - `특수문자 포함`(파일명 안전) · `공백 포함` · `Backspace로 마지막 글자 지우기`
