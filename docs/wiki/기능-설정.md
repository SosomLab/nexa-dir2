# 설정

설정 창은 VS Code식으로 사이드바 카테고리 + 검색 + 즉시 적용을 제공합니다.

![설정 창](https://raw.githubusercontent.com/wiki/SosomLab/nexa-dir2/images/20-preferences.png)

- **진입**: `Ctrl+,` · 파일 메뉴 `설정...` · 도구 모음 설정(⚙) 버튼.
- **검색**: 상단 검색 상자에 입력하면 좌측 카테고리 트리가 라벨/설명 매치로 필터링됩니다(다중 토큰 AND·`{N}개 일치`·✕ 지우개).
- **즉시 적용**: 대부분의 옵션은 바꾸는 즉시 적용·저장됩니다(저장 버튼 없음, 비정상 종료 시 유실 방지).

## 카테고리별 옵션

| 카테고리 | 옵션 |
| --- | --- |
| **모양 (Appearance)** | 테마(시스템/라이트/다크), 숨김 파일 표시, 점(.) 파일 표시, 하단 도크 표시 |
| **글꼴 (Fonts)** | 기본 글꼴, 파일 목록 글꼴(+크기), 폴더 이름 굵게, 컬럼 헤더 굵게/이탤릭, 상태바 글꼴, 우클릭 메뉴 글꼴, 대화상자 글꼴(+크기 7~24) |
| **언어 (Language)** | 표시 언어(시스템/내장·설치 언어) |
| **파일 목록 — 보기·정렬** | 폴더 우선 정렬, 대소문자 구분 정렬, 위로 이동 시 선택 위치(상단/중단/하단), 컬럼 자동 맞춤 최대 폭(50~2000), 도구 모음 순서[편집…], 파일 컬럼[편집…] |
| **타입어헤드 (Type-ahead)** | 검색 범위(전체/현재 계층/가시 스트림), 입력 리셋(200~10000ms), 배지 위치(3×3), 특수문자 포함, 공백 포함, Backspace로 마지막 글자 지우기 |
| **컨텍스트 메뉴 (Context Menu)** | 컨텍스트 메뉴 항목[편집…] |
| **탭 (Tabs)** | 탭 더블클릭 동작(탭 닫기/고정/잠금) |
| **하단 도크 (Bottom Dock)** | 하단 도크 표시 |
| **터미널 (Terminal)** | 터미널 글꼴(폴백 체인)·크기(8~32), 줄 바꿈, 열 수(80~1000) |

## 테마

- **시스템 / 라이트 / 다크**. 시스템은 OS 설정을 실시간 추종(레지스트리 변경 감지). DWM 다크 타이틀바 지원.
- **조작**: 보기 메뉴 라디오 또는 `F6` 순환. 영속.

## 언어 (i18n)

- 내장 **한국어·영어** + exe 옆 `data\lang\`에 키 단위 오버라이드. `시스템`은 OS 언어를 추종.
- **동적 전환**: 재시작 없이 언어가 바뀝니다(테이블 스왑 + 메뉴/컬럼 재구성 + 재그리기).

## 글꼴 슬롯

용도별로 글꼴을 따로 지정합니다: 기본(메뉴·탭·경로·설명·도크) / 파일 목록(+헤더 공유·폴더 굵게·헤더 굵게/이탤릭) / 콘솔(터미널) / 상태바 / 우클릭 메뉴(예약) / 대화상자. 설치된 글꼴을 자기 글꼴로 렌더하는 드롭다운으로 고릅니다.

## 영속 위치

모든 설정은 **exe 옆 `data\` 폴더**에 저장됩니다(레지스트리·%APPDATA% 비의존). 쓰기 불가한 위치면 `%LOCALAPPDATA%\NexaDir\data`로 폴백합니다.

- `settings.cfg` — 앱 설정 전반
- `session.cfg` — 화면 세션(패널·탭·펼침)
- `data\lang\` — 사용자 언어팩
- `data\renames\` — 일괄 이름 변경 프리셋
