# 파일 목록과 보기

파일 목록은 Nexa Dir 2의 심장입니다. 100k 항목도 첫 렌더 115ms·스크롤 60fps로 다루는 가상화 리스트 위에, 3가지 보기 모드와 강력한 컬럼 시스템이 얹혀 있습니다.

![파일 목록](https://raw.githubusercontent.com/wiki/SosomLab/nexa-dir2/images/01-main-overview.png)

---

## 보기 모드 3종 (트리 / 플랫 / 타일)

파일 목록 표시 방식을 3가지로 전환합니다.

| 모드 | 설명 |
| --- | --- |
| **트리 보기** | 폴더를 목록 내 제자리에서 펼침/접힘(별도 트리 패널 없이 평면 스트림). 기본값. |
| **플랫 보기** | 계층 없는 일반 목록. |
| **타일 보기** | 아이콘 32px 그리드. "내 PC"에서는 드라이브 용량 바 표시. |

- **전환**: 보기 메뉴의 라디오(`트리 보기`/`플랫 보기`/`타일 보기`) 또는 도구 모음의 view 그룹 3버튼.
- **영속**: 보기 모드는 **탭별 상태**로 세션에 저장됩니다. 새 탭은 현재 모드를 계승합니다.

---

## 가상화 리스트

가시 영역의 행만 커스텀 드로잉하는 가상 리스트입니다. 100k 행 첫 렌더 115ms, 스크롤 2.1ms/프레임을 달성합니다. 휠·키보드 스크롤 모두 지원하며, 프레임 비용이 창 높이에만 비례합니다.

---

## 컬럼 시스템

기본 5개 컬럼: **이름 · 확장자 · 크기 · 수정한 날짜 · 종류**. 말줄임 트리밍, 가로 스크롤, 타임존 반영 날짜를 지원합니다. "내 PC" 루트에서는 `전체 크기`·`사용 가능한 공간` 컬럼이 자동 노출됩니다.

### 컬럼 리사이즈 · 재배열 · auto-fit

- **리사이즈**: 헤더 경계를 드래그.
- **드래그 재배열**: 헤더를 잡고 이동하면 고스트 헤더가 커서를 따라오고, 경계 근처에서 삽입 미리보기가 표시됩니다. `Esc`로 취소.
- **auto-fit**: 컬럼 경계를 **더블클릭**하면 내용 너비에 맞춰 조정됩니다(최대 폭 = 설정 `컬럼 자동 맞춤 최대 폭`, 기본 400px).

### 표시/숨김 · 순서

설정 → 파일 목록 → `파일 컬럼`의 [편집…]에서 컬럼별 표시/숨김과 순서를 별도 창에서 바꿉니다(이름 컬럼은 잠금·전부 숨김 방지). 변경은 세션에 저장됩니다.

---

## 정렬 (다중열)

- 헤더 클릭 = 3상태 순환(오름차순 → 내림차순 → 해제).
- **`Shift`+클릭 = 다중열 정렬**. 화살표(▲▼)는 앞, 순번(①②)은 뒤에 표시됩니다(순번은 상시 표시).
- **관련 설정**:
  - `폴더 우선 정렬` — 정렬 방향과 무관하게 폴더를 파일보다 앞에 모읍니다(기본 켜짐).
  - `대소문자 구분 정렬` — 대문자로 시작하는 항목을 위 그룹으로.

---

## 인라인 펼침

트리 보기에서 폴더를 목록 내 제자리에서 펼치고 접습니다(별도 트리 패널 없음).

- **조작**: ▸/▾ 마커 클릭 · 캐럿 위치에서 `→`(펼침) / `←`(접힘).
- **핵심**: 삼각형 마커(=펼침)와 행 본문(=선택)이 분리되어 있어, 펼친 상태에서 서로 다른 폴더의 항목을 동시에 선택하는 **폴더 교차 다중 선택**(플래그십)이 가능합니다. → [파일 조작](기능-파일-조작)의 선택 참고.
- 펼침 상태는 탭별로 세션에 영속됩니다.

---

## 컬럼 너비 동기화

듀얼 파일 패널에서 좌/우 컬럼 폭을 실시간 동기화합니다.

- **조작**: 보기 메뉴 `컬럼 너비 동기화` 또는 도구 모음 `colsync`(⇔) 버튼(듀얼일 때만 활성).
- **동작**: 끄면 패널별 독립, 다시 켜면 포커스 패널 기준으로 맞춰집니다. 기본 켜짐·영속. 탭은 패널 폭을 상속합니다.
