# 일괄 이름 변경

여러 파일의 이름을 규칙적으로 한 번에 바꿉니다. **동작 카드를 파이프라인으로 쌓아** 위에서 아래로 순서대로 적용하고, 실시간 미리보기로 결과를 확인합니다.

![일괄 이름 변경 창](https://raw.githubusercontent.com/wiki/SosomLab/nexa-dir2/images/30-bulk-rename.png)

- **진입**: `Ctrl+Shift+R` 또는 편집 메뉴 `일괄 이름 변경...`. 선택한 항목이 대상입니다.

---

## 동작 파이프라인 (6종)

동작 카드를 `+`/`−`로 추가·삭제하고 순서대로 적용합니다(마지막 1장은 삭제 불가).

| 동작 | 설명 |
| --- | --- |
| **치환 (텍스트)** | 찾기/바꾸기. 모드(모든/첫/마지막/전체 교체)·대소문자 일치. |
| **치환 (정규식)** | 정규식 기반 치환. |
| **대소문자 변경** | 유지 / 대문자 / 소문자 / 단어별 / 문장형. |
| **텍스트 삽입** | 접두/접미 위치에 텍스트 삽입. |
| **연번 붙이기** | 시작·증가·자릿수·위치. |
| **날짜 삽입** | 수정한/만든 날짜·포맷 토큰(`${YYYY}-${MM}-${DD}` 등). |

---

## 적용 대상 (스코프)

각 동작은 이름의 어느 부분에 적용할지 지정할 수 있습니다: **이름 / 이름+확장자 / 확장자 / 확장자(점 포함)**.

---

## 실시간 미리보기

- 우측 그리드에 **이전 → 이후**가 실시간으로 표시됩니다.
- 하단에 `{N}개 항목이 변경됩니다` 건수 라벨.
- **충돌 표시**: 빈 이름 / 금지 문자 / 중복 / 이미 존재.
- 행별로 제외 토글이 가능합니다.
- **[이름 변경]** 버튼은 충돌이 0이고 변경이 1건 이상일 때만 활성화됩니다.

---

## 적용 · 되돌리기

적용은 하나의 트랜잭션(`MoveBatchOp`)으로 처리되어, `Ctrl+Z` 한 번으로 배치 전체를 되돌릴 수 있습니다.

---

## 프리셋

자주 쓰는 동작 순서를 저장·재사용합니다.

- 좌하단 **[… ⌄]** 메뉴 = 저장된 프리셋 목록 / `저장` / `편집`.
- `저장`: 현재 파이프라인을 이름 붙여 저장(`data\renames\*.cfg`).
- `편집`: 프리셋 목록에서 행별 삭제.
