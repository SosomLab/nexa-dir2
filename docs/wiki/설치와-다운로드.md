# 설치와 다운로드

Nexa Dir는 **포터블 단일 exe**가 기본이고, **설치형 exe**를 보조로 함께 제공합니다.
아래 방법 중 하나를 고르면 됩니다.

| 방법 | 대상 | 상태 |
| --- | --- | --- |
| **winget — 포터블** | 명령 한 줄로 설치·업데이트하고 싶을 때 | ✅ **제공 중** |
| **GitHub Release — 포터블 exe** | 다운로드해서 바로 실행(설치 없음) | ✅ 제공 중 |
| **GitHub Release — 설치형 exe** | 시작 메뉴 등록·제거 관리자를 원할 때 | ✅ 제공 중 |
| winget — 설치형 / Chocolatey | — | ⏳ 심사 중(승인 후 제공) |

> **시스템 요구사항**: Windows 10 (버전 1809, 빌드 17763) 이상 · **x64**.

---

## 1. winget으로 설치 (포터블 · 권장)

Windows 10/11에 기본 포함된 **winget**(App Installer)으로 한 줄이면 됩니다.

```powershell
winget install SosomLab.NexaDir.Portable
```

- 설치 후 어디서든 **`nexadir`** 명령으로 실행됩니다(PATH에 자동 등록).
- 업데이트:

  ```powershell
  winget upgrade SosomLab.NexaDir.Portable
  ```

- 제거:

  ```powershell
  winget uninstall SosomLab.NexaDir.Portable
  ```

> ⚠️ **포터블 제거 시 데이터도 함께 삭제됩니다.** winget 포터블은 exe를
> `%LOCALAPPDATA%\Microsoft\WinGet\Packages\` 아래 고정 폴더에 두고, 설정·세션이
> 담기는 `data\` 폴더도 그 안에 쌓입니다. `winget uninstall`이 이 폴더를 통째로
> 지우므로 설정·세션이 같이 사라집니다. 데이터를 보존하려면 제거 전에 해당 폴더의
> `data\`를 백업하세요.

---

## 2. GitHub Release에서 직접 다운로드

패키지 매니저 없이 파일을 바로 받고 싶다면 **[Releases](https://github.com/SosomLab/nexa-dir2/releases/latest)**
에서 최신 버전을 받습니다.

### 포터블 exe (설치 불필요)

- 자산: **`NexaDir-<버전>-win-x64.exe`** (약 1.5MB)
- 다운로드 후 **더블클릭하면 바로 실행**됩니다. 설치·레지스트리·관리자 권한이 필요 없습니다.
- 설정·세션은 exe **바로 옆에 `data\` 폴더**로 저장됩니다. USB나 폴더째 다른 PC로
  옮겨도 그대로 동작합니다.
  (exe 위치가 쓰기 불가한 경로면 `%LOCALAPPDATA%\NexaDir\data`로 자동 폴백합니다.)

### 설치형 exe (시작 메뉴·제거 관리자)

- 자산: **`NexaDir-Setup-<버전>.exe`** (약 2.6MB)
- 실행하면 설치 마법사가 뜹니다. 시작 메뉴 바로가기와 **프로그램 추가/제거** 항목이 등록됩니다.
- 설치형은 **제거해도 사용자 데이터(`data\`)를 보존**합니다(포터블과 다른 점).

> 🔐 현재 바이너리는 **코드 서명이 없습니다.** SmartScreen이 "알 수 없는 게시자"
> 경고를 띄우면 **추가 정보 → 실행**으로 진행하세요. 무결성은 Release 자산의
> SHA-256으로 확인할 수 있습니다.

---

## 3. 어떤 걸 골라야 하나

- **가볍게 써보고 싶다 / USB로 들고 다닌다** → 포터블(winget 또는 Release exe).
- **명령으로 설치·업데이트를 관리하고 싶다** → **`winget install SosomLab.NexaDir.Portable`**.
- **시작 메뉴 등록과 제거 관리자를 원한다** → 설치형 exe.

포터블과 설치형은 기능이 동일합니다. 차이는 **배치 방식과 제거 시 데이터 처리**뿐입니다.

---

## 4. 준비 중인 채널

- **winget 설치형**(`SosomLab.NexaDir`) — microsoft/winget-pkgs 등록 심사 중.
- **Chocolatey**(`nexa-dir`, `nexa-dir.portable`) — 커뮤니티 모더레이션 심사 중.

승인되면 이 페이지에 설치 명령을 추가합니다. 그 전까지는 위의 **winget 포터블** 또는
**GitHub Release** 를 이용하세요.
