# 파일 조작

복사·이동·삭제·이름 변경·새로 만들기와, 이를 뒷받침하는 선택·클립보드·드래그앤드롭·되돌리기까지 정리합니다.

---

## 선택

### 폴더 교차 다중 선택 (★ 플래그십)

트리 인라인 펼침 상태에서 **서로 다른 부모 폴더의 항목을 동시에 선택**할 수 있습니다. 삼각형(=펼침)과 본문(=선택)이 분리되어 있어 가능한 동작입니다.

| 조작 | 동작 |
| --- | --- |
| 본문 클릭 | 단일 선택 |
| `Ctrl`+클릭 | 선택 토글(추가/제거) |
| `Shift`+클릭 | 앵커 기준 범위 선택 |
| `Ctrl+A` | 전체 선택 |
| 빈 영역 드래그 | 러버밴드(사각 범위) 다중 선택 |

키보드 캐럿 내비게이션은 → [키보드 단축키](키보드-단축키) 참고.

## 복사 / 이동 (전송 엔진)

단일 전송 경로로 처리됩니다. 같은 폴더 규칙(이동=무동작, 복사=" (2)" 순번 복제), 4MB 청크 진행률, 동일 볼륨 fast path, 순환(자기/하위) 금지를 지원합니다.

- **조작**: `Ctrl+C`/`Ctrl+X` → `Ctrl+V`, 또는 드래그앤드롭.
- **붙여넣기 대상**: `Ctrl+V`는 **폴더를 1개 선택**한 상태면 **그 폴더 안**으로, **파일을 1개 선택**한 상태면 **그 파일이 있는 폴더**로 붙여넣습니다(트리 펼침으로 하위 폴더의 파일을 선택한 경우 보이는 최상위가 아니라 그 폴더). 선택이 없거나 다중 선택이면 현재 폴더입니다.
- **잘라내기 표시**: `Ctrl+X`한 항목은 목록에서 **흐리게** 표시됩니다. 탐색기에서 잘라낸 파일도 동일하게 흐려지며, 복사/붙여넣기 등으로 클립보드가 바뀌면 자동 해제됩니다(OS 클립보드 단일 출처).
- **진행 창**: `전송 {N}%` 표시, 완료 시 `[닫기 (N)]` 카운트다운 버튼.
- **충돌 처리**: 충돌 항목만 순차로 `덮어쓰기`/`모두 덮어쓰기`/`건너뛰기`/`취소`(선택은 남은 충돌 전체에 적용). `Esc` = 전송 취소.

## 이름 변경 · 새로 만들기

- **이름 변경**: `F2`(또는 느린 재클릭)로 인라인 오버레이 편집. 문자·Backspace·`Enter` 확정·`Esc` 취소·`Ctrl+C/X/V`.
- **새로 만들기**: `Ctrl+Shift+N`=새 폴더, 파일 메뉴 `새 파일 만들기`. 생성 후 즉시 인라인 리네임.
- 여러 파일의 규칙적 이름 변경은 → [일괄 이름 변경](기능-일괄-이름변경).

## Undo / Redo

이동/복사/이름변경/새로 만들기를 되돌립니다(스택 2개·상한 100).

- **조작**: `Ctrl+Z` / `Ctrl+Y`(또는 `Ctrl+Shift+Z`), 편집 메뉴.
- **휴지통 복원**: 휴지통 삭제도 되돌릴 수 있습니다(undelete 동사). 단 **완전 삭제는 되돌릴 수 없습니다**(설계).

## 클립보드 · 드래그앤드롭

- **클립보드**: OS 클립보드 단일 출처(CF_HDROP). 탐색기 ↔ 앱 사이 `Ctrl+C/X/V` 왕복. 잘라내기는 1회성이며 대기 중 항목은 **흐리게** 표시됩니다.
- **드래그앤드롭(OLE)**: 
  - 수신 — `Ctrl`=복사 / `Shift`=이동 / 기본=볼륨별 규칙. 자기/하위로의 드롭 금지.
  - 발신 — 행을 누른 뒤 임계치 이상 이동하면 드래그 시작. 이동이어도 원본은 안전하게 보존되는 방향.
  - **엣지 자동 스크롤** — 드래그 중 목록 위/아래 가장자리(한 행 높이)에 커서를 대면 목록이 연속 스크롤됩니다.
  - **호버 전환/펼침** — 드래그 중 **다른 탭 위에 3초**(기본) 머물면 그 탭이 활성화되고, **접힌 폴더 이름 위에 3초** 머물면 폴더가 펼쳐집니다(펼쳐진 안쪽 폴더에 다시 머물면 중첩 펼침 — 실수로 접히지는 않습니다). 대기 시간은 [설정](기능-설정)의 **파일 전송 → 드래그 호버 전환 시간**(200~10000ms, 기본 3000)으로 조절합니다.

## 폴더 watcher (자동 반영)

현재 폴더가 외부에서 바뀌면(생성·삭제 등) 300ms 디바운스로 무간섭 재로드합니다. 펼침·선택·캐럿·스크롤 위치가 보존됩니다. 실패 시 `F5`로 수동 새로 고침할 수 있습니다.
