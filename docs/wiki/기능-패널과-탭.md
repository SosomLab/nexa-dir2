# 패널과 탭

Nexa Dir는 좌우 두 개의 **파일 패널**과, 각 패널의 독립 **탭**으로 여러 위치를 동시에 다룹니다.

---

## 싱글 / 듀얼 파일 패널

- **듀얼**: 좌우 2개 패널(파일 관리자 커맨더 규약 — 한쪽에서 다른 쪽으로 복사/이동).
- **싱글**: 우 패널 레이아웃을 숨김. 우 패널의 상태(탭·위치)는 보존되어 듀얼로 복귀하면 원복됩니다.
- **조작**: 보기 메뉴 `듀얼 파일 패널`/`싱글 파일 패널` 라디오, 또는 도구 모음 `panel toggle`(▌▐) 버튼.
- **연동**: 싱글 파일 패널에서는 정보 패널이 싱글로 고정됩니다(상태바 안내). → [하단 도크](기능-하단-도크)

---

## 패널별 탭

각 패널에 독립적인 탭 바가 있습니다(활성 탭 accent 강조·× 닫기·[+] 버튼). 각 탭은 독립된 뷰 + 독립된 히스토리를 가집니다.

| 조작 | 동작 |
| --- | --- |
| `Ctrl+T` | 새 탭 |
| `Ctrl+W` | 탭 닫기 |
| `Ctrl+Tab` | 활성 패널의 다음 탭 |
| `Tab` | 활성 패널 전환(좌 ↔ 우) |
| `[+]` 클릭 / 탭 바 빈 공간 더블클릭 | 새 탭 |

---

## 탭 잠금 / 고정 / 복제

탭을 **우클릭**하면 메뉴가 나옵니다.

| 항목 | 동작 |
| --- | --- |
| 탭 잠금 / 해제 | 닫기에서 제외(실수 방지) |
| 탭 고정 / 해제 | 탭 고정(📌) |
| 탭 복제 | 같은 위치의 새 탭 |
| 탭 닫기 | (잠겼거나 탭이 1개면 비활성) |

- **탭 더블클릭 동작**은 설정에서 지정합니다(`탭 더블클릭 동작` — 닫기 / 고정 / 잠금).

---

## 세션 복원

앱을 다시 켜면 이전 작업 상태가 그대로 복원됩니다.

- 복원 대상: 패널별 탭 목록·활성 탭·펼침 상태·정렬 모드·컬럼 표시/폭·활성 패널.
- 탭이 바뀌면 1초 디바운스로 자동 저장됩니다(코얼레싱 — 잦은 변경을 묶어 flush).
- 저장 위치: exe 옆 `data\session.cfg`.
