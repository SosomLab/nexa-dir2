  대상 = **편집 중 경로바**(활성 패널 우선), 캐럿 x = 필드 pad + `text_width(버퍼)`(끝 고정 α 모델).
  결과 문자열은 DefWindowProc의 WM_IME_CHAR→WM_CHAR 변환으로 **기존 문자 경로 그대로 수신**.
- `PathBar::edit_info()` 신설(버퍼·필드 rect·pad 노출). `Win32_UI_Input_Ime` feature + B3에
  imm32.dll 등재.
- **실기**: 편집 모드 진입(우클릭) → WM_IME_* 주입 + 한글 SendKeys("가나다") → 생존·Esc 복귀·정상
  종료 확인. 실제 한국어 IME 조합 창 위치는 수동 확인 항목(자동화 불가 — IME 활성화 스크립팅 제약).
