> macOS 오버레이(스크롤 시 얇은 썸·페이드, 썸 드래그 시 일반 바) — **가로도
> 동일** ③ 프리셋 메뉴 = [저장된 프리셋들/─/Save Renaming Sequence…/Edit
> Renaming Sequences…] + Save 이름 입력 팝업.

- **NxGrid(ctl 14호)**: 컬럼 헤더+경계 드래그 리사이즈(±4px·COL_MIN 40) ·
  체크 마크 열(행별 `Option<bool>` — AA 체크·클릭 토글 NXGR_TOGGLE/GETROW) ·
  `set_rows` 전체 교체 API. 미리보기 적용(행별 제외 excluded — 카운트 연동).
