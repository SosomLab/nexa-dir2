## colsync SVG 교체 — 파서 fill·text 확장 (main 직커밋)

> **사용자 제공 SVG**: 컬럼 동기화 = 채움 양방향 화살표 + "SYNC" 텍스트 —
> 기존 스트로크 서브셋 밖이라 파이프라인 확장.

- **svg.rs 확장**: 루트 `fill` 모드(색 지정 = 채움·none/부재 = 스트로크) +
  `<text>` 요소(x/y[베이스라인]/font-size/font-weight/text-anchor=middle —
  본문은 태그 뒤 조각에서 수집). 테스트 +2(56 green).
- **gdipctx**: 채움 = `GdipFillPath`(도형)·텍스트 = `GdipAddPathString`
  아웃라인 패스(Arial 고정·중앙 앵커 = StringFormat center·베이스라인 근사).
- **icons**: `colsync` SVG 4호 등록 + **`-disabled` 변형 = 원본 SVG를 알파
  38% 잉크(SVG_INK_DIM)로 렌더** — disabled PNG 관례 폐지. colsync PNG
  6종(활성 3+비활성 3) 제거. 실기: 활성 SYNC·싱글 모드 흐림 캡처 검증.
